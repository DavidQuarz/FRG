# FRG

## Préparation
- Créer son environnement virtuel et l'activer.

```bash
python3 -m venv venv
source venv/bin/activate
```
- Installer les prérequis

```bash
pip install -r requirements.txt
```

- Définir la variable d'environnement

```bash
FLASK_APP=frg.py
```

## Lancer Flask

```bash
flask run
```
